# Проект — соответствие кода UML-диаграмм

Реализация является точным и полным воплощением представленной диаграмме классов. Инкапсуляция соблюдена через использование классов. Наследование и полиморфизм реализованы через `abc` и переопределение абстрактных методов. Связи реализованы через хранение ссылок на объекты в атрибутах класса.
Сделать программу по этим диаграммам классов возможно. Поэтому моя реализация полностью ей соответствует. Второй лабораторной у меня не было, чтобы сказать есть ли между кодам и ER-диаграммами соответствие. Интерфейс тоже реализуем, хотя и есть некоторые вопросы, например: 'что за начать расследование'.
